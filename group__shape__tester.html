The classes in mra/sdf_shape_3D.h illustrate how to define surfaces as MADNESS functions for subsequent use in calculations using them as sources or for enforcing boundary conditions on interior surfaces. This example instantiates each shape and plots it for subsequent visual verification (with OpenDX).
